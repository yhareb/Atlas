# Atlas Project Instructions

## Permanent standing rules

### Message length limit
If your response to any prompt requires more than 4 Telegram messages, you MUST compile the full response into a single Markdown file and send only the file path or attachment. Never send more than 4 Telegram messages in a single reply, regardless of how long the content is.

This rule applies to all responses — work order confirmations, audit reports, investigation results, everything.
